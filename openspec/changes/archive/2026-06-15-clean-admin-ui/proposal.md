# 提案：清理后台管理端前端页面广告并更换系统名称

## 问题描述 (Problem Description)
后台管理端（admin-frontend）的首页和打包配置中包含大量原作者（若依）的宣传广告、捐赠二维码、QQ群、Gitee链接以及版本更新日志等信息，并且系统名称默认为“若依管理系统”。这不符合我们当前的“星空集市控制台”项目定位，需要进行清理和定制化修改。

## 根本原因分析 (Root Cause Analysis)
前端代码直接采用了开源的若依Vue前端模板，保留了大量原版的默认页面内容和配置文本（主要集中在 `package.json`、`.env`配置文件、首页组件 `src/views/index.vue`、侧边栏标题、页面标题 `src/layout/components/Sidebar/Logo.vue` 等位置）。

## 修复目标 (Fix Goal)
1. 清除 `src/views/index.vue` 中的所有无用广告、联系方式、更新日志、捐赠等信息，替换为简洁的数据概览或欢迎文案。
2. 将系统中所有“若依管理系统”及相关字样统一替换为有创意的名称，暂定为“星空集市控制台”。
3. 确保页面正常运行，无相关错误和功能缺失。
